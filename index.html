Kantonio Brownlee's Final Integration Project

(Color Circle Palette Generator)

This project is a cute and simple project that creates circles that have matching color aesthetics to the word selected.
For example, if you pick the word 'soft', once you click the joystick many differnt circles filled woth soft colors will appear.
Then the output of the arduino board will be a screen that says the name of the word chosen.
Along with the circles, a tone.js sound will be made while the joystick is pressed.
